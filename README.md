Система управления библиотекой

Простое десктопное приложение на Java (Swing) с графическим интерфейсом и хранением данных в SQLite.

Функции
- Добавление/удаление книг и читателей
- Выдача и возврат книг
- Поиск по названию и автору
- Отслеживание просрочек

Как запустить
1. Убедитесь, что установлена Java (JDK 17+)
2. Скачайте JAR-файл из раздела Releases (или скомпилируйте сами)
3. Запустите: `java -jar LibraryApp.jar`
